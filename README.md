# juanpzuluaga.github.io

Personal site of Juan Pablo Zuluaga. Built with [Hugo](https://gohugo.io/) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod), deployed to GitHub Pages via Actions on every push to `master`.

## Local

```bash
git clone --recurse-submodules git@github.com:JuanPZuluaga/JuanPZuluaga.github.io.git
hugo server          # http://localhost:1313
```

## Edit

- Home, currently, about: `layouts/index.html`
- Publications: `data/publications.yaml`
- Projects: `content/projects/*.md`
- CV: `content/cv.md`
- Contact: `content/contact.md`
- Theme tokens / styling: `assets/css/extended/theme.css`
